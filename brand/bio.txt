Name:
Nexus Systems

Short tagline:

Discord Developer Portal "Description" field (character-limited, ~400 max):
Nexus Systems is a full community security & moderation bot with a real
web dashboard. AutoMod, anti-raid, tickets, verification, leveling,
giveaways, reaction roles, and more — every setting configurable per
server, right from the browser. Built and maintained by CH Services.

Longer bot description (for listing sites / README-style use):
Nexus Systems is an all-in-one Discord bot for community security and
moderation, built by CH Services. It combines the moderation depth of a
traditional bot (AutoMod, anti-raid, warnings, temp-bans, mod logs) with
engagement tools (leveling, starboard, giveaways, tickets, suggestions)
and a complete web dashboard — so server staff can configure everything
visually instead of memorizing commands. Every server gets its own
independent configuration, permission teams, and command controls, making
Nexus Systems suitable for anything from a small community to a large,
multi-team server network.
